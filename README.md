# ☁️ Simple Weather App

A sleek, responsive weather application built with Vanilla JavaScript that fetches real-time data from the OpenWeatherMap API.


## ✨ Features
* **Real-Time Weather:** Get live temperature, humidity, and wind speed for any city.
* **Dynamic Icon Mapping:** The app automatically switches icons (Rain, Clouds, Clear, etc.) based on real-time conditions.
* **Smart UI:** Temperatures are rounded for better readability, and units (km/h, %) are appended automatically.
* **Dual Search:** Search by clicking the icon or pressing the "Enter" key.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Modern layout with a glassmorphism-inspired gradient.
* **JavaScript (ES6+):** Uses `Async/Await` for API calls and Object Mapping for icon logic.
* **API:** [OpenWeatherMap API](https://openweathermap.org/)

## 🧠 Lessons Learned
* Implementing **Asynchronous JavaScript** to handle network requests without freezing the UI.
* Using **Object Mapping** to replace long `if-else` chains, making the code more maintainable.
* Managing **DOM Manipulation** to update the interface dynamically based on JSON responses.
